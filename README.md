### Core Commands:
- `setup`: Configure your AI Neural Core (Gemini, OpenAI, Claude, or Perplexity).
- `scan <url>`: Initialize a full-spectrum adversarial audit on a target.
- `scan file <path>`: Perform an AI-powered neural audit (SAST) on a local file.
- `report`: View the deduplicated Vulnerability Matrix and PoC dashboard.
- `ai`: Trigger the Red Team Lead AI for multi-stage attack chaining analysis.
- `export <file>`: Save all telemetry and findings to a JSON file.
